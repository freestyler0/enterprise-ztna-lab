\# Network Device Configurations



This directory contains sanitized Cisco device configurations used in the Enterprise Zero Trust Architecture lab.



\## Devices



\- Cisco ISR 2911 Router

\- Cisco Catalyst 2960S Switch

\- Cisco ASA 5520 Firewall

&#x20;

\## Purpose



The configurations demonstrate the network architecture implemented in the lab, including:



\- Router-on-a-Stick (802.1Q)

\- VLAN segmentation

\- DHCP services

\- NAT

\- Transparent firewall deployment

\- Layer 2 switching

\- SSH management



\## Disclaimer



All configuration files have been sanitized for public release.



The following information has been removed or replaced:



\- Passwords and secrets

\- Serial numbers

\- Certificates

\- Environment-specific identifiers

\- WAN addressing
