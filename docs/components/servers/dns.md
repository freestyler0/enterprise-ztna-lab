# DNS Infrastructure

Microsoft DNS provides centralized name resolution for the enterprise environment.

The DNS service is integrated with Active Directory and is responsible for:

- Resolving internal hostnames
- Supporting Active Directory services
- Name resolution for application servers
- Service discovery
